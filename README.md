# Modelado-ARCH-GARCH-de-la-Inflaci-n-del-IPC-de-Malta
Estudio de la inflación del IPC en Malta utilizando modelos ARIMA para la estructura de la media y GARCH para la volatilidad condicional. El análisis incluye pruebas de estacionariedad, identificación del modelo, diagnóstico de residuos, detección de heterocedasticidad y generación de pronósticos.
